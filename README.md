# 💸 Smart Expense Tracker (Multi-UI Project)

A modern **Smart Expense Tracker** web application with **5 UI styles**, **theme switching**, and **analytics dashboard**.

> 🚀 Built using HTML, CSS, JavaScript (No Backend Required)

---

## 🌟 Features

✨ Add, Edit, Delete Expenses  
📊 Category-wise Analytics (Chart.js)  
🎨 5 UI Styles (Switch anytime):
- Modern Minimal (Apple-like)
- Neon / Gradient (FinTech)
- Material UI (Google-like)
- Dashboard Professional
- Glassmorphism

🌗 Theme Modes:
- Light Mode
- Dark Mode
- System Default

📄 Export data to CSV  
📑 Pagination (5 records per page)  
💾 LocalStorage (No database required)

---

## 🖥️ UI Preview

### 🎨 Multiple UI Styles
- Switch UI using button (Top-right)
- Real-time UI change (No reload)

### 🌙 Theme Switching
- Toggle between Light / Dark / System
